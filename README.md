# Student Information Board

## Project Overview
This project is a full-stack student information board web application. It allows students and admins to view and manage announcements, events, timetables, and results. The backend is built with Node.js, Express, and SQLite. The frontend is a simple HTML/CSS/JS app.

## Features
- User registration and login (admin and student roles)
- CRUD operations for announcements, events, timetables, and results (admin only)
- Search and filter functionality
- Archive feature for old content
- Local SQLite support

## Prerequisites
- Node.js (v16+ recommended)
- npm
- SQLite
